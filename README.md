## 项目定位
专为普通人设计的一站式、互动式金融投资入门课程。基于“教学练用”闭环，以第一性原理重新拆解金融常识。

## 学习理念 (教学练用闭环)
1. **教 (Teach) - 认知重塑:** 避免晦涩术语，从常识和图形化数据切入（如复利曲线、通胀侵蚀）。
2. **学 (Learn) - 互动反馈:** 测验、拖拽概念图，随时巩固概念。
3. **练 (Practice) - 场景模拟:** 给定特定历史时期（如2008年次贷危机、2020年熔断），让用户做出选择并看结果。
4. **用 (Use) - 实战落地:** 根据用户的财务状况、风险偏好，利用API或算法自动生成一份基础的资产配置方案（Mock Portfolio）。

## 核心课程模块设计
### Module 1: 财富的本质与时间的朋友 (Time Value of Money)
- 第一性原理：钱的时间价值（通胀与复利）。
- 互动：动态调节年化收益率和定投金额，可视化看到“复利”后期的爆发力量。

### Module 2: 世界上没有免费的午餐 (Risk & Return)
- 第一性原理：风险与收益成正比。收益是对承担风险的补偿。
- 互动：滑动条调整“期望收益”，展示对应的“历史最大回撤（Max Drawdown）”，理解波动。

### Module 3: 市场买的是什么？ (Asset Classes - 债与股)
- 第一性原理：债是借钱（拿利息），股是合伙（分利润）。
- 互动：资产类别卡片拖拽分类（高风险/低风险，固定收益/权益类）。

### Module 4: 鸡蛋与篮子 (Portfolio & Diversification)
- 第一性原理：非相关性资产的组合可以降低非系统性风险。
- 互动：“沙盘推演”：全仓股票 vs 股债平衡，遇到大跌时的净值表现对比。

## 技术栈与媒介选择
- **前端:** React (Vite) + TypeScript
- **样式:** Tailwind CSS
- **可视化/交互:** Recharts (展示收益曲线), Framer Motion (页面平滑过渡与卡片交互)
- **图标:** Lucide React
- **路由:** React Router (单页面互动课程流)

## 数据接口 (如需API)
- 若需要拉取真实标的（如标普500，沪深300的历史数据），可使用 AlphaVantage, Yahoo Finance API 等。本项目现阶段内置历史典型年份的Mock数据用于“练”与“用”环节，保证学习闭环完整流畅。
